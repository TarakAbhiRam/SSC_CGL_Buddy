Help: what files work?

PDF import

  • Use digital PDFs where text can be selected.
  • Each question needs exactly four options.
  • Each question needs an answer marker like Ans. (b), Answer: B, or Correct answer: 2.
  • Scanned PDFs should be converted/imported page-by-page as images.

Image import

  • Use PNG, JPG, WEBP, BMP, or GIF files.
  • Clear screenshots work better than angled photos.
  • Uses Gemini vision and may hit Gemini free-tier quota.
  • Imported questions appear in Database → Browse questions.
